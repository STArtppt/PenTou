# PenTou —— 你的本地优先 AI 对话管理器

## 🌟 什么是 PenTou？

在与各类 AI 大模型（如 ChatGPT, Claude, Gemini 等）高频对话的今天，我们的知识和灵感往往散落在各个平台独立的对话框中，难以统一检索和归档。

**PenTou** 是一款专为高效知识管理而生的**本地优先 AI 对话管理器**。它致力于帮助用户将散落的 AI 对话内容进行离线汇总、检索与深度再加工，让你真正掌握自己与 AI 共同创造的数字资产。

## ✨ 核心特性

### 1. 💬 全局对话管理
不再受限于不同平台的历史记录。PenTou 支持统一导入和管理你的 AI 对话数据，让所有对话在一个清爽、快速的界面中集中呈现。通过强大的检索与左侧栏快捷批量管理功能，随时找回过往的灵感火花。

### 2. 📄 深度文档加工与提炼
仅仅保存对话是不够的。PenTou 引入了创新的**文档再加工**机制。你可以从冗长的对话记录中提取有价值的内容（例如通过消息右侧的“摘录”功能），将其转化为结构化的独立文档。无论是代码片段、长篇文章还是学习笔记，都能以最直观的形式沉淀在你的本地知识库中。

### 3. 🔒 本地优先，数据掌控
你的数据完全属于你。PenTou 采用基于本地文件系统（Markdown）的数据存储方式，所有的对话数据和知识文档都保存在你的设备上，告别隐私泄露焦虑与云服务锁定的风险。

### 4. 🐳 极客友好的私有化部署
对于高阶用户，PenTou 提供了完善的 Docker 部署支持与轻量的登录鉴权机制。你可以轻松地将它部署在个人的云主机或家中 NAS 上，随时随地安全地访问你的个人私有知识库。

## 🎯 适合哪些人？

- **AI 重度依赖者**：每天都在和各种大模型交互，希望将有价值的对话统一留存。
- **知识工作者/创作者**：需要从 AI 讨论中提炼文章、灵感、策划案的用户。
- **开发者**：经常利用 AI 辅助编程，需要整理和归档各种技术方案与代码片段。
- **数据隐私倡导者**：希望将重要知识库和数据完全掌握在自己手中的数字极客。

## 🚀 技术追求

PenTou 追求极致的交互体验与简洁的架构。应用前端采用 React + Vite + Tailwind CSS 构建现代化 UI；后端则以创新的自研 Vite 插件提供轻量级服务，确保数据原生落地为清晰易读的 Markdown 文件。
